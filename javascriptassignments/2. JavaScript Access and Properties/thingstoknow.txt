Know keywords and existing functions of JavaScript?
JavaScript Keywords
var: Declares a variable (function-scoped).
let: Declares a block-scoped variable.
const: Declares a block-scoped, read-only constant.
function: Declares a function.
return: Exits a function and optionally returns a value.
if, else, switch, case, default: Conditional statements.
for, while, do: Looping statements.
break, continue: Controls loop flow.
try, catch, finally, throw: Error handling.
class: Declares a class.
extends, super: Used with classes for inheritance.
import, export: Module import and export.
async, await: Asynchronous programming.
new: Creates a new instance of an object.
this: Refers to the current object context.
delete: Deletes an object property.
typeof: Returns the type of a variable.
instanceof: Tests if an object is an instance of a class.
JavaScript Existing Functions
alert(message): Displays an alert dialog with a message.
console.log(value): Logs information to the console.
parseInt(string, radix): Converts a string to an integer.
parseFloat(string): Converts a string to a floating-point number.
Number(value): Converts a value to a number.
String(value): Converts a value to a string.
Array.isArray(value): Checks if a value is an array.
Object.keys(object): Returns an array of an object's property names.
Object.values(object): Returns an array of an object's property values.
Object.entries(object): Returns an array of an object's key-value pairs.
Math.max(...values): Returns the largest of zero or more numbers.
Math.min(...values): Returns the smallest of zero or more numbers.
Date.now(): Returns the current timestamp in milliseconds.
